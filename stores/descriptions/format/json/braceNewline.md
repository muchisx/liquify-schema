&nbsp;⚙️&nbsp;&nbsp;&nbsp;**Default** `false`

&nbsp;💁🏽‍♀️&nbsp;&nbsp;&nbsp;Recommended setting is `false`

#### Brace Newline

The `braceNewline` rule will insert newlines at the top and bottom of nested properties.

#

---


#### 👍 &nbsp;&nbsp; Disabled

```json

{
 "one": {
    "xx": {
      "xx": false
    }
  },
  "two": {
    "xx": {
      "xx": false
    }
  }
}


```

---

#### 👎 &nbsp;&nbsp; Enabled

```json

{
 "one": {

    "xx": {

      "xx": false

    }

  },
  "two": {

    "xx": {

      "xx": false

    }

  }
}


```
